# 创建博客系统


### 安装cnpm (如果有的话请忽略)

> npm install -g cnpm --registry=https://registry.npm.taobao.org

### 初始化项目

> cnpm i

### 编写博客

source/_posts/下编写 markdown文件，
文件开始写入

'''  
    ---   
      title: 测试   
      tags: java  
      categories: 编程  
    ---  
'''

### 生成项目

> hexo g

### 启动项目

> hexo s

### 访问

[本地博客](http://locolhost:4000)

### 清除命令
> hexo clean
